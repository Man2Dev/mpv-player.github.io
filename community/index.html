Community and contacts

There are several ways by which you can engage with the mpv community of users and developers. Here's the complete list:

Internet Relay Chat

This is the preferred way to discuss mpv use and development. Never ask to ask - just state your question. Stick around until someone replies.

Users channel #mpv on irc.freenode.net
Development channel (no fun allowed) #mpv-devel on irc.freenode.net

GitHub issue tracker

This is the preferred way to report bugs.

Issue tracker https://github.com/mpv-player/mpv/issues

Private contact

If you write to this e-mail only members of the mpv team will be able to read it. Please use it only for comunication that must be private. Use IRC or the issue tracker for anything else.

Private contact e-mail mpv-team@googlegroups.com
